# 科学计算的矩阵类

### 前言：写这个项目主要是因为现代密码学课程设计要实现一些密码算法。作为一个喜欢面向对象编程的人，当然要写数据结构啦。思来想去还是矩阵类最合适，毕竟很多密码都有线性部件，而进行线性运算用矩阵来当容器真是再合适不过。

## 项目简介：

- 实现numpy中的array大部分操作

- 实现matrix中大部分操作

- 使用多线程编程实现
